# VowusProject
# 泛亚电竞-专业竞技联盟

  应用介绍: "竞技宝-专业竞技联盟"app是一款预约VR/网咖和购买VR/网咖商品的应用。

  核心功能: 预约+商城。
  
    商城：VR眼镜、网咖(耳麦、鼠标)；
         核心流程：线上下单 - 凭订单编号 - 到店取货。
          
    场馆：VR/网咖馆；
          预定后需在预约日期到达对应场馆进行消费，逾期订单将自动取消。
          
    首页：热门商品、热门场馆、精选场馆、精选商品等。
    
    订单：商品订单、我的预约等。
    
    消息：订单消息、系统消息。
